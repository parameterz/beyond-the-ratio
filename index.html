The left ventricular mass (LVM) is calculated using the following equation: LVM = 0.8 * (LV end-diastolic volume) + 0.6 * (LV end-systolic volume) - 0.2 * (LV end-diastolic area) This equation is used to estimate LVM from echocardiographic measurements.
